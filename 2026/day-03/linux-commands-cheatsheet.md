Process management
ps	Displays information about running processes.	
top	Shows real-time system and process activity.	
htop	Interactive process viewer (if installed).	
kill	Terminates a process using its PID.
pkill	Kills processes matching a name or pattern.	

File system
pwd	Displays the current working directory.	
ls	Lists files and directories.	
cd	Changes the current directory.
mkdir	Creates a new directory.	
rmdir	Removes an empty directory.	
rm	Deletes files or directories.	
cp	Copies files or directories.	
mv	Moves or renames files/directories.	
touch	Creates an empty file or updates its timestamp.	
cat	Displays the contents of a file.

Networking troubleshooting

ip addr	Displays IP addresses and network interfaces.	
ip link	Shows network interface status.	
ip route	Displays the routing table.	
ping	Tests connectivity to a host.	
traceroute	Shows the path packets take to a destination.	
tracepath	Traces the network path without root privileges.	
nslookup	Queries DNS to resolve a domain name.
dig	Performs detailed DNS lookups.
